# Theory Documentation

This directory contains the mathematical foundations and physical theory
underlying the NeuroWave electromagnetic simulation framework.

## Contents (Planned)

| Document | Description |
|----------|-------------|
| `maxwell_equations.md` | Maxwell's equations in differential and integral forms |
| `yee_algorithm.md` | The Yee FDTD algorithm derivation |
| `cfl_condition.md` | Courant-Friedrichs-Lewy stability analysis |
| `numerical_dispersion.md` | Numerical dispersion error analysis |
| `pml_theory.md` | Perfectly Matched Layer theory |
| `dispersive_materials.md` | Debye, Drude, Lorentz material models |
| `tissue_dielectrics.md` | Biological tissue dielectric properties |

## References

1. A. Taflove and S. C. Hagness, *Computational Electrodynamics: The Finite-Difference Time-Domain Method*, 3rd ed., Artech House, 2005.
2. K. S. Yee, "Numerical solution of initial boundary value problems involving Maxwell's equations in isotropic media," *IEEE Trans. Antennas Propag.*, vol. 14, no. 3, pp. 302-307, 1966.
3. J.-P. Berenger, "A perfectly matched layer for the absorption of electromagnetic waves," *J. Comput. Phys.*, vol. 114, pp. 185-200, 1994.
